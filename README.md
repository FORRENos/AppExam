# AppExam

WPF-приложение на C# для модулей 2-3 демоэкзамена.

Платформа проекта:

```text
.NET Framework 4.8
```

## Что есть в проекте

- авторизация по ролям;
- гостевой вход;
- просмотр списка товаров из БД или демо-данных;
- поиск по товарам;
- сортировка по цене и остатку;
- фильтр по размеру скидки;
- добавление, редактирование и удаление товаров для администратора;
- подсветка товаров со скидкой больше 25%;
- подсветка товаров, которых нет на складе.

## Как открыть

Открыть в Visual Studio 2022 файл:

```text
AppExam.sln
```

## База данных

Файл базы данных не хранится в репозитории. Приложение сначала пробует подключиться к SQL Server,
а если база недоступна, запускается на демо-данных из кода.

Основная строка подключения находится в файле:

```text
App.config
```

По умолчанию используется сервер:

```text
(localdb)\MSSQLLocalDB
```

Если на компьютере другой сервер SQL Server, поменяйте `Data Source` в `App.config`, например:

```text
Data Source=.\SQLEXPRESS
```

## Основные файлы

- `MainWindow.xaml` - окно авторизации.
- `AdminWindow.xaml` - окно каталога товаров.
- `Windows/ProductEditorWindow.xaml` - окно добавления и редактирования товара.
- `Models/Product.cs` - класс товара.
- `Models/User.cs` - класс пользователя.
- `Services/SqlDataStore.cs` - работа с SQL Server.
- `Services/DemoDataStore.cs` - демо-данные, если БД не подключилась.
- `Services/ProductViewManager.cs` - поиск, фильтрация и сортировка.
- `Services/ProductFactory.cs` - копирование данных товара при редактировании.
- `App.config` - строка подключения к базе данных.
